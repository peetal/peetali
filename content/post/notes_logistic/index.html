5. Softmax regression as neural network

Softmax can also be seen as a neural network with only a single layer of neuron, with the number of neurons in this layer being the number of total classes. Each neuron in this layer has its own linear part, where the weight matrix wiw_iwi​﻿ and offset bib_ibi​﻿ are unique. The outputs of these neurons would go into a non-linear activation function (can be viewed as the generalized version of the sigmoid function). This activation function ggg﻿ is determined by mapping the natural parameter of the exponential family to the canonical parameter that parameterizes the probability distribution of a single multinomial trial. The predicted y^\hat{y}y^​﻿ is a nnn﻿-vector where n equals the total number of neurons/classes.
